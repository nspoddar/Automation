It is a simple java project created for testing https://developer.here.com/documentation page.

Pre-requisites
/** TOOLS
	-Eclipse IDE
	-JDK installed
	-TestNG
**/

-Download the project from this link
-Open the eclipse and import this project 
-Required jars are added under 'jars' folder in the project structure.
Go to project build path and add all these as external jar files
-I have used the chrome driver, and kept it under 'drivers' folder.
-Execute this java program as TestNG
